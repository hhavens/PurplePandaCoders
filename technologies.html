Technologies I Enjoy

The tools and languages I use to bring my project ideas to life

Swift & SwiftUI

I love creating iOS and macOS applications with Swift. It's a powerful yet approachable language that makes app development enjoyable. SwiftUI's declarative syntax makes building interfaces intuitive and fun.

What I enjoy:

  • Building native iOS apps
  • SwiftUI's declarative approach
  • The Swift ecosystem and community

HTML, CSS & JavaScript

Building websites and web applications is my creative outlet. I enjoy bringing designs to life with HTML, CSS, and JavaScript. The instant visual feedback of web development is particularly satisfying.

What I enjoy:

  • Creating responsive designs
  • CSS animations and transitions
  • Interactive JavaScript features

PowerShell

Automating tasks with PowerShell is both practical and satisfying. I create scripts to streamline my digital workflow, manage files, and automate repetitive tasks. The ability to control Windows systems programmatically is powerful.

What I enjoy:

  • Automating repetitive tasks
  • System administration scripts
  • File and folder management
