For Users

  1. Open the App Store on your iOS device
  2. Search for "Card Rewards Guru"
  3. Tap "Get" to download and install
  4. Open the app and create an account
  5. Start adding your credit cards

For Developers

# Clone the repository
git clone https://github.com/yourusername/card-rewards-guru.git
cd card-rewards-guru

# Install dependencies
npm install

# Install iOS dependencies
cd ios && pod install

# Run on iOS simulator
npm run ios

# Run on Android emulator
npm run android

ü§ù Contributing

We welcome contributions! Here's how you can help:

Reporting Bugs

  1. Check if the bug has already been reported
  2. Create a new issue with a descriptive title
  3. Include steps to reproduce the bug
  4. Add screenshots if applicable
